BioTaxonomy

A learning application for browsing animals and plants according to their taxonomy. It can be used by biology students, secondary and higher education teachers, students, agronomists, veterinarians or generally anyone interested in biology.
The application is implemented in Html and jQuery / Javascript. Data is received in JSON format with SPARQL queries to RDF database DBpedia and using the API of Wikipedia.

  • Date: October 15, 2015
  • Technologies: Javascript, jQuery, Bootstrap, SPARQL
  • Employer: FOSS Centers of Excellence
  • Github page: github.com/ellak-monades-aristeias/BioTaxonomy
  • Live View: ellak-monades-aristeias.github.io/BioTaxonomy

TinyCollab

TinyCollab is a collaborative text editor where two or more users can make changes to text files in real time and communicate with each other through chat. The synchronization of users is implemented using sock.js, a javascript library that simulates websockets. Usercake is used for user management. TinyCollab was developed as a contribution work for the second school code of Excellence Unit of the University of Crete held from 08.07.2015 - 14.07.2015 and was one of the open source projects that received an award.

  • Date: July 14, 2015
  • Technologies: jQuery, WebSockets, php, node.js
  • Github Page: github.com/faysvas/TinyCollab
